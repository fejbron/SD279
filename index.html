Learning Objectives
01
Understand fundamental data structures including arrays, lists, stacks, queues, trees, graphs and hash tables
02
Design and implement efficient algorithms for searching, sorting and manipulation of data structures
03
Perform complexity analysis using asymptotic notation (Big-O, Ω, Θ) to compare algorithmic efficiency
04
Apply abstract data type concepts and distinguish between interface and implementation
05
Use mathematical models of data — sets, lists, graphs — to solve computational problems
06
Identify application areas for each data structure and algorithm in real-world software systems

Lecture Materials

Slide Decks

Week 2 & 4
Intro to DSA & Arrays
What are data structures and algorithms? Abstract Data Types, complexity recap, categories of DS, and a deep dive into the Array ADT — memory layout, operations, patterns.
25 slides
↗
Week 3
Asymptotic & Complexity Analysis
Big-O, Big-Ω, Big-Θ, little-o and little-ω. Formal proofs, loop analysis rules, nested loops, recurrence relations, Master Theorem, and amortised analysis.
40 slides
↗
Week 5
Stacks & Queues
LIFO and FIFO principles, push/pop/enqueue/dequeue operations, array and linked list implementations, balanced brackets, BFS, deque, priority queue, and classic interview problems.
25 slides
↗
Week 6
Linked Lists
Singly, doubly and circular linked lists. Traversal, insert, delete operations with visual traces. Floyd's cycle detection, reversal, find middle, merge sorted lists.
25 slides
↗
Week 7
Hash Maps
Hash functions, collision handling (chaining vs open addressing), load factor, rehashing, LRU cache, hash sets, consistent hashing, and common patterns like two-sum and anagram grouping.
25 slides
↗
Week 8
Binary Trees & BSTs
Tree terminology, binary tree types, node structure, DFS traversals (inorder, preorder, postorder), BFS level-order, BST property, search/insert/delete, balancing and complexity.
18 slides
↗
Week 10
Graphs
Graph theory, directed vs undirected, weighted graphs, adjacency matrix and list representations, DFS, BFS, cycle detection, topological sort, Dijkstra's, Kruskal's MST.
25 slides
↗
Week 11–12
Sorting & Searching Algorithms
Bubble, Selection, Insertion, Merge, Quick, Heap sort. Counting and Radix sort. Linear search, Binary search and its variants. Full complexity table and real-world implementations.
28 slides
↗

Key Policies

Important Guidelines

Quizzes
Unannounced pop-up quizzes. No make-ups for missed quizzes. One lowest quiz score will be dropped at end of semester.
Late Work
Submissions lose 40% per late day. One day late = 60% max. Two days late = 20% max. Submit before the deadline.
Integrity
All code must be your own. Copying or sharing code results in failing the project and referral to HOD. Discuss bugs but never share solutions.
Submission
Submit as a zip file named with your ID. Include README, source code, test files, and documentation explaining your solution.
Office Hours
Office hours are for clarifying lecture content or resolving serious problems — not for re-running missed lectures or getting assignment answers.
Attendance
Attendance counts for 10 marks. Class participation counts for 5. Regular attendance and engagement are essential for success.
